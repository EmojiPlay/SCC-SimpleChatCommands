SCC - SimpleChatCommands

SimpleChatCommands is a Garry's Mod addon for your chat

With this addon you will don't need to open Q menu and seek for "Guns" category, and then seek for gun that you need.

You can just type "!pistol" in chat and BOOM, there is a pistol in your inventory from Half-Life 2 game.

If you want to try it out you can just find this addon on steam workshop or GitHub

Full list of commands is on GitHub Wiki
